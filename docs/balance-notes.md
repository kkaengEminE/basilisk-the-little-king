# 밸런스 메모 (Balance Notes)

> 나중에 한 번에 검수할 때 참고할 튜닝 관찰 기록. 수치는 코드가 출처(여기는 "왜")이며,
> 플레이 검증에서 발견한 의사결정과 의도를 남긴다.

## Phase 1 (Forest 코어 루프)

- **포식 루프가 처음엔 작동 안 함**: 자동 플레이 봇이 25초간 먹이 0마리 → 수탉에 사망.
  먹이가 너무 멀리서/빠르게 도망쳐서 따라잡아도 못 먹음.
  - 해결: 바실리스크에 **내재 마그넷(포식 견인) `magnet: 115`** 도입(뱀서바류 자동 수집),
    `pickupRange: 7`, 먹는 반경 `head+6`, `FLEE_RADIUS 150→115`, 석화 먹이도 마그넷에 끌리게.
  - 결과: 봇 기준 38초에 9마리 포식·길이 17·레벨3·진화1·만피 생존(LOOP_OK).
- **초반 수탉이 너무 치명적**: 음파/접촉 `13/16 → 9/10`, 속도 `52→46`, 첫 등장 `12s→18s`.
- 이동 `150→155`, 재생 `0.5→0.8`.

## Phase 2 (거울/물 반사 · 꼬리 슬램)

- **거울 반사가 매우 치명적**: 시선 1회당 `gazeDamage 26 × reflect 0.9 ≈ 23` 자해.
  쿨다운 1.1s마다 명중 시 약 5초 만에 사망(헤드리스 실측).
  - 의도: 신화 충실성상 "거울을 노려보면 자멸"은 강해야 함. 다만 실플레이에서 과하면
    아래 레버로 완화: `MIRROR_BEARER.mirrorReflect`(현 0.9), 물 `reflect 0.35`,
    `reflectResist` 업그레이드 스택(+0.2, 상한 0.8), 거울이 콘에 "들어오자마자"가 아니라
    중심 판정이라 스치듯 지나가면 안 맞음.
  - **TODO(폴리시)**: 첫 반사 때 경고 연출/짧은 무적 유예 고려.
- 꼬리 슬램: 물 흐림 6초 / 거울 덮기 5초. 넉백 260(먹이는 ×0.8, 피해 없음).

## Phase 3 (인간 적군)

- 적 종류별 의도:
  - **사냥꾼(hunter)** hp28: 거리 285 유지 카이팅, 2.2s마다 화살(속도 250, 피해 11).
    근접 위협은 낮지만 다수면 누적 피해 큼. 시선으로 쉽게 석화.
  - **기사(knight)** hp120, **gazeResist 0.6**: 시선 피해 40%만 들어감 → 독/꼬리 권장.
    접촉 피해 18로 가장 아픔. 정면 돌격.
  - **사제(priest)** hp48: 2.4s마다 반경170 내 아군 +16 치유. 먼저 처치 1순위.
  - **수탉 조련사(handler)** hp52: 5.5s마다 수탉 소환(전체 수탉 3마리 상한).
- 인간은 기본적으로 시선에 석화됨(기사만 장갑으로 일부 저항). 거울잡이 거울에 가린 적은 반사.
- 맵별 인간 구성: Forest=사냥꾼, Farm=사냥꾼+조련사, Monastery=사제+사냥꾼,
  Castle=기사+사냥꾼+사제, Kingdom=전부. (수치는 `world/maps.ts`)
- **균형 주의(미검증)**: 화살은 현재 저항 무시(직격 11). 후반 다수 사냥꾼이 가혹할 수 있음 →
  필요시 `arrowDamage`/`humanCap`/`humanEvery` 조정. 기사 gazeResist 0.6이 너무 단단하면 0.4~0.5로.

## Phase 5 (폴리시)

- **거울 반사 완화**: `MIRROR_BEARER.mirrorReflect` 0.9 → **0.7**(거울잡이 거울).
  물 웅덩이 0.35는 유지. 전체 자살급 반사를 한 단계 누그러뜨림.
- 사운드/연출/저장은 게임플레이 수치에 영향 없음(품질만). 사운드 전체 음소거 = `M`.
- **죽음 판정 주의(정상 동작)**: `updatePlaying`에서 regen(이동 시스템) → 피해(충돌) → `hp<=0`
  판정 순서. 같은 틱에 치명타→판정이라 정상 사망하지만, 외부에서 `hp=0`만 찍으면
  다음 틱 regen이 되살림. 자동 테스트로 죽일 땐 `stats.regen=0`도 함께 설정할 것.

## Phase 6 (보스·확장)

- **족제비 보스(Weasel)**: hp 420, 평소 96·러지 320(0.45s 버스트, 2.4s 주기), 접촉 24, xp 120.
  **gazeImmune**(신화상 바실리스크 천적) → 독/꼬리/회피로만 처치. 스테이지 진행 62% 시점 1마리 스폰.
  보스 머리 위 체력바, 등장/처치 배너. **균형 주의(미검증)**: 후반 맵에서 다른 적과 겹치면 가혹할 수
  있음 → `WEASEL.hp`/`lungeSpeed`/스폰 시점(`map.duration*0.62`) 조정 레버.
- **신규 업그레이드**: `armor`(Stone Skin, +12%/스택, 상한 0.6, 모든 피해 감소) ·
  `lifesteal`(Carrion Feast, 처치당 +4 회복). 둘 다 defense 카테고리.
- armor는 출처 저항(reflect/rooster) 곱한 뒤 추가로 곱연산: `dealt = amount*(1-resist)*(1-armor)`.
